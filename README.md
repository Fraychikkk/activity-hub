\# ActivityHub



ActivityHub — учебный pet-проект для практики QA и автоматизации тестирования.



Сервис позволяет пользователям создавать активности и искать участников для совместных мероприятий: спорт, настольные игры, прогулки, клубы и другие события.



\## Цель проекта



Показать навыки:

\- анализа требований;

\- написания тестовой документации;

\- API-тестирования;

\- автоматизации тестирования;

\- работы с GitHub;

\- базовой backend-разработки.



\## Стек



\- Python

\- FastAPI

\- SQLite

\- Pytest

\- Postman

\- GitHub Actions
